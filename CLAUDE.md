# Projekt szabályok

- Magyar nyelvű kommentek a kódban
- Hibakezelés try-catch blokkal minden aszinkron műveletnél
- Minden függvénynek legyen JSDoc dokumentációja
- Maximális fájlméret: 300 sor
- Használj érthető változóneveket
- Kerüld a console.log használatát production kódban
- Minden pull request-hez szükséges a code review
